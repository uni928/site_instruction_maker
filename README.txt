Site Instruction Maker - Chrome拡張（フルページ撮影版 ver1.3.0）

■ 使い方
1. chrome://extensions を開く
2. 右上の「デベロッパーモード」をON
3. 「パッケージ化されていない拡張機能を読み込む」を押す
4. このフォルダを指定する

■ 機能
- フルページ撮影して開始: 現在のタブを上から下まで自動スクロールし、フルページ画像を作って編集画面を開きます。
- 表示範囲だけ撮影: 従来どおり今見えている範囲だけ撮影します。
- 前回の内容から再開: IndexedDBに自動保存された内容を再開します。
- インポートして再開: 以前エクスポートしたJSONを読み込んで編集を再開します。
- 画像上をクリック: 番号マーカーを追加します。
- マーカーをドラッグ: 位置を移動します。
- 選択中マーカーの青いハンドルをドラッグ: 線の角度を回転します。
- Delete: 選択中のマーカーを削除します。
- Ctrl+S: JSONをエクスポートします。
- 出力画像を作成: 上に注釈入り画像、下に各番号の説明を並べたPNGを出力します。

■ 保存形式
- エクスポート: JSON
- 最終出力: PNG

■ 注意
- フルページ撮影中は、固定ヘッダーや固定ボタンなどの重複しやすい要素を一時的に非表示にします。
- とても長いページでは、ブラウザのメモリやキャンバス上限の影響で失敗する場合があります。
- 20を超える番号は丸数字ではなく通常数字表示になります。

- 下部の説明一覧は、内容の行数を計算して可能な限り横方向へ詰めて配置します（段組み配置）。


ver1.3.4: 説明欄では、textareaの外側の灰色部分をクリックしたときだけ対応する数字を選択します。
